> Small and fast UUID (unique id) generator.

### How to Install

```sh
$ npm install rkgttr-uuid --save-dev
```
or

```sh
$ yarn add rkgttr-uuid --dev
```

### Getting Started

```js
var uuid = require('rkgttr-uuid');

var id = uuid();
```
or

```js
import uuid from 'rkgttr-uuid';

let id = uuid();
```
